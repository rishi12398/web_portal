What is PPC?

PPC stands for Peripheral DMZ Cloud. It is a multi-cloud hybrid Infrastructure which supports secure hosting of Demo / POC and Enterprise Applications. Hosting will be on public cloud platform such as AWS and Microsoft Azure.

Microsoft Azure cloud

Microsoft Azure is a cloud computing service created by Microsoft for building, testing, deploying, and managing applications and services through a global network of Microsoft-managed data centers.
Refer Link for more details - https://azure.microsoft.com/en-in/overview/what-is-azure/

Amazon web services cloud

Amazon AWS is a cloud computing service created by Microsoft for building, testing, deploying, and managing applications and services through a global network of Amazon-managed data centers.
More details on AWS, refer link - https://aws.amazon.com/what-is-aws/

Google cloud

Google Cloud Platform, offered by Google, is a suite of cloud computing services that runs on the same infrastructure that Google uses internally for its end-user products, such as Google Search and YouTube. Alongside a set of management tools, it provides a series of modular clous services including computing, data storage, data analytics and machine learning. Google Cloud Platform provides laaS, PaaS, and Serverless computing environments. More details on GCP, refer link - https://cloud.google.com/docs/overview/
